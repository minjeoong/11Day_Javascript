# JavaScript_Basic

makeIntro(name, mbti) :

form에 입력한 이름과 MBTI 문자열이 주어졌을 때,
이름과 Mbti 를 포함해서 문자열로 return.


assign2(arr, divisor) :

arr는 정수가 담겨있는 배열
divisor은 양의 정수
arr의 각 원소 중 divisor로 나누어 떨어지지 않는 값들을 배열에 담아 return.
단, 이때 반환하는 배열은 오름차순으로 정렬해서 return.
만약 나누어 떨어지지 않는 값이 단 한개도 없다면 배열에 "없어요!" 만 담아서 return.
